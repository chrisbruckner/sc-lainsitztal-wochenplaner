🎓 KEY TECHNICAL DECISIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Decision: Webcal over API
  → Decentralized, no authentication needed
  → Automatic format (RFC 5545 iCalendar)
  → Each team has independent URL
  
Decision: Vanilla JS + HTML5
  → No framework overhead
  → Fast load times
  → Easy to understand & modify
  
Decision: LocalStorage for Manual Games
  → Browser-side persistence
  → No database needed (MVP phase)
  → Automatic sync with Webcal
  
Decision: html2canvas for Export
  → Client-side generation
  → No server processing
  → User controls download
  
Decision: 5-Minute Cache
  → Balance between freshness & performance
  → Reduces ÖFB server load
  → Configurable if needed
